# Chrome Web Store listing — HuePick

**Name:** HuePick — Color Picker & Palette
**Summary (132 char):** Pick any colour on screen, build palettes, copy HEX/RGB/HSL and check WCAG contrast. Local-first, no ads, no tracking.
**Category:** Developer Tools
**Single purpose:** Let users pick colours from the screen and manage a colour palette.
**Permission justification — storage:** Saves the user's palette locally on their device. No other data is accessed or transmitted.
**Data usage:** Does not collect or transmit any user data. Palette stored only in chrome.storage.local.
